技术架构

编程语言: Rust 2021 Edition
UI框架: egui + eframe
异步运行时: tokio
HTTP客户端: reqwest
音频处理: rodio + cpal
配置管理: config + toml
日志系统: log + env_logger
错误处理: thiserror + anyhow

核心模块:
- src/tts/ - TTS客户端模块
- src/ai/ - AI文案生成模块
- src/soundboard/ - 音效板模块
- src/audio/ - 音频路由模块
- src/ui/ - 用户界面模块
- src/config/ - 配置管理模块
- src/utils/ - 工具函数模块
